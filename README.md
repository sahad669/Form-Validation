# Form Validation Project

Live Site: [https://sahad669.github.io/Form-Validation/](https://sahad669.github.io/Form-Validation/)

## Description

This is a simple and responsive form validation project built using HTML, CSS, Bootstrap, and JavaScript.

The form validates user input before submission. It checks:
- Required fields
- Email format using Regular Expressions (Regex)
- Password strength using Regex

## Features

- Real-time input validation
- User-friendly error messages
- Responsive design with Bootstrap
- Client-side validation only (no backend)

## Technologies Used

- HTML
- CSS
- Bootstrap
- JavaScript (Vanilla)

## How to Use

1. Visit the [Live Site](https://sahad669.github.io/Form-Validation/)
2. Fill in the form fields (email and password are required).
3. Submit the form to see validation in action.
